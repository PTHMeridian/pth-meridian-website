SA'AT Cryptographics — Open Source

The cryptographic stack

Three open-source tools. One coherent platform. From raw key generation to self-sovereign identity with zero-knowledge proofs.

// 01
AKR KeyGen
ASYMMETRIC KEY RESOLVER — KEY GENERATION ENGINE

Post-quantum cryptographic primitive engine implementing ML-KEM, ML-DSA and SLH-DSA. NIST FIPS 203/204/205 compliant. Hybrid classical/quantum modes, REST API and live developer dashboard. The foundation everything else is built on.

ML-KEM-768 ML-DSA-65 SLH-DSA ECDSA P-256 / P-384 AES-256-GCM Hybrid modes REST API TypeScript
↗
// 02
AKR Naos
NESTED AUTHENTICATION OPERATIONS SUITE

A six-phase cryptographic trust platform with seven hardening steps. Shamir Secret Sharing, Argon2id key vault, Certificate Authority with OCSP, secure channel, HMIT audit engine, identity wallet and zero-knowledge proofs with bounds-binding range proofs and Merkle membership proofs.

Shamir Secret Sharing Argon2id KDF ZKP v2 PKI / OCSP HSM Bridge Threshold Signatures Key Attestation DID / Wallet

APPLIED MINDSHIFT UNIVERSAL NAVIGATOR

A consequence mapping tool that surfaces second and third-order effects before decisions become irreversible. Helps individuals and organizations navigate complexity by making the invisible visible — the downstream consequences most decision-makers never see coming.

About PTH Meridian

Canadian-built.
Globally distributed.

PTH Meridian is an independent cryptographic research and development studio based in Calgary, Alberta, Canada. We build open-source post-quantum security infrastructure grounded in NIST FIPS 203, 204 and 205 — designed to be independently auditable, academically rigorous and production-ready.

Our goal is to contribute to Canadian and international post-quantum cryptographic standards — building the civilian infrastructure that keeps organizations, individuals and institutions secure through the quantum transition and beyond.

We are actively seeking academic collaboration with Canadian university cryptography research groups for formal security review of our zero-knowledge proof and post-quantum protocol implementations.
